# 盈利宝理财项目

## 一、项目简介
这是一个大型分布式金融理财项目。  
1、后端YLB-server部分采用的技术栈包括:SpringBoot,Spring Task,Dubbo,zookeeper,mybatis,mysql,redis等；  
2、前端YLB-client部分采用技术栈包括:vue,axios,JavaScript,Promise等；  
3、项目采用微服务的架构，包括多个独立的微服务，micr-api接口、micr-common公共服务、micr-web服务、micr-database数据库服务、micr-pay支付服务、micr-task定时任务服务、ready-mapper逆向工程；
